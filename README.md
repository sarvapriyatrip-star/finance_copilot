# 💼 Finance Copilot

Finance Copilot is an AI-powered finance assistant built using Python, Streamlit, LangChain and Google's Gemini model.

## Features

- AI-powered finance assistant
- Finance-focused responses
- Conversation memory
- Clean Streamlit interface

## Technologies

- Python
- Streamlit
- LangChain
- Gemini 2.5 Flash

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
